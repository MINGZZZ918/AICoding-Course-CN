# AI Coding 小白课程

> **零基础，用 AI 把想法变成产品**

这是一套面向完全零基础小白的 AI 编程入门课程。你不需要任何编程基础，只需要会打字、会描述想法，就能用 AI 做出自己的小程序、网页和浏览器插件。

## 🎯 课程特点

- **零门槛**：不需要任何编程基础
- **实战导向**：边学边做，直接上手
- **完整项目**：从零做一个微信小程序并发布上线
- **授之以渔**：学会和 AI 协作的方法，举一反三
